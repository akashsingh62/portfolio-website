## 🚀 Deploy on GitHub Pages (Step-by-Step)

### Step 1 — Create a GitHub Repository
1. Go to [github.com](https://github.com) and log in
2. Click **New Repository**
3. Name it: `akash-portfolio` (or anything you like)
4. Set it to **Public**
5. Click **Create Repository**

### Step 2 — Upload the File
1. In your new repo, click **Add file → Upload files**
2. Drag and drop `index.html` into the upload area
3. Write a commit message: `Initial portfolio upload`
4. Click **Commit changes**

### Step 3 — Enable GitHub Pages
1. Go to your repo's **Settings** tab
2. Scroll to **Pages** in the left sidebar
3. Under **Source**, select `Deploy from a branch`
4. Choose branch: `main` | Folder: `/ (root)`
5. Click **Save**

### Step 4 — Done! 🎉
Your site will be live in ~60 seconds at:
```
https://<your-github-username>.github.io/akash-portfolio
```
